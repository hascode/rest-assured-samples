Basic Authentication
Request URL Method Content-Type Extras

HTTP Headers
Request URL Method Content-Type Extras
/service/header/print GET - Header "myparam"
Response Status Content-Type Body/Content
200 text/plain String value of given header param "myparam"
Returning multiple HTTP Headers
Request URL Method Content-Type Extras
/service/header/multiple GET - -
Response Status Content-Type Headers
200 text/plain customHeader1:foo, anotherHeader:bar
Sending Cookies
Request URL Method Content-Type Extras
/service/access/cookie-token-secured GET - Cookie: authtoken
Response Status Content-Type Body
403 (if cookie authtoken != abcdef)/ 200 text/plain -
Receiving Cookies
Request URL Method Content-Type Request Parameter
/service/cookie/modify GET - name
Response Status Content-Type Cookie
200 text/plain userName:value-of-name-request-parameter

Response Status Content-Type Body
